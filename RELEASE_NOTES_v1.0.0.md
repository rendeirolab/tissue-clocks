# tissue-clocks v1.0.0

Source code release accompanying the Nature Medicine manuscript:

**Histological aging signatures for monitoring tissue-specific aging and disease**

## Contents

- Full analysis pipeline for GTEx histopathology aging clocks
- Revision analyses: foundation models, external cohort validation, ICC reproducibility, Bland-Altman calibration, GNN models, DNA methylation comparison
- Didactic walkthrough notebook: `src/revision/tissue_clock_walkthrough.ipynb`
